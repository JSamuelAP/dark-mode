Modo oscuro

Applicación web que simula el modo oscuro de Twitter. Además de poder cambiar entre modo claro y oscuro, se puede elegir un color principal. La primera vez que lo utillizas el tema es seleccionado según las preferencias del navegador, cuando el tema y el color son configurados, esta información se almacena en el local storage, por lo que tus preferencias permaneceran aunque se cierre la página.

Esta aplicación fue desarrollada con:

  • HTML
  • CSS
  • Materialize CSS
  • JavaScript
